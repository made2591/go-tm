## Installation

Just get the library by:

```sh
go get github.com/made2591/go-tm
```

## Usage

To run a [BB-2 Game](https://en.wikipedia.org/wiki/Busy_beaver), from inside the ```root``` folder of the project run

```sh
go run main.go
```
